02

Définition et premières propriétés

Définition par l'intégrale

La fonction logarithme népérien \(\ln\) est l'unique primitive de \(x\mapsto\dfrac{1}{x}\) sur \(\mathbb{R}^*_+\) qui s'annule en 1 : \[\ln x = \int_1^x \frac{1}{t}\,\mathrm{d}t\quad (x>0)\] Cette définition garantit immédiatement que \(\ln' = 1/x\) et \(\ln 1 = 0\).

Définition par la réciproque de l'exponentielle

\(\ln\) est la bijection réciproque de \(\exp\) :

\[\forall x\in\mathbb{R},\; e^{\ln x} = x \quad(x>0)\qquad \forall y>0,\; \ln(e^y) = y\] En particulier : \(\ln e = 1\), \(\ln 1 = 0\), \(\ln(e^2)=2\), \(\ln(\sqrt{e})=1/2\).

Domaine, variations et limites

• Domaine : \(\mathbb{R}^*_+\)

• \(\ln' (x) = 1/x > 0\) → strictement croissante sur \(\mathbb{R}^*_+\)

• \(\lim_{x\to+\infty}\ln x = +\infty\) ; \(\lim_{x\to 0^+}\ln x = -\infty\) (AV \(x=0\))

• \(\ln 1 = 0\) → la courbe passe par \((1;0)\)

• \(\ln e = 1\) → la courbe passe par \((e;1)\)

Valeurs remarquables

Propriétés algébriques

Pour \(a>0\), \(b>0\), \(n\in\mathbb{Z}\), \(\alpha\in\mathbb{R}\) : \[\ln(ab) = \ln a + \ln b\] \[\ln\!\left(\frac{a}{b}\right) = \ln a - \ln b\] \[\ln(a^n) = n\ln a\] \[\ln\!\left(\sqrt{a}\right) = \frac{1}{2}\ln a\] \[\ln(a^\alpha) = \alpha\ln a\]
Démonstration — ln(ab) = ln(a) + ln(b)
Soit \(g(x)=\ln(ax)\) pour \(x>0\) (avec \(a>0\) fixé).
\(g'(x)=\dfrac{a}{ax}=\dfrac{1}{x}=(\ln x)'\)

Donc \(g\) et \(\ln\) ont la même dérivée → \(g(x)=\ln x + C\) pour une constante \(C\).
En \(x=1\) : \(g(1)=\ln a = \ln 1 + C = C\).
Donc \(\ln(ax) = \ln x + \ln a\), c'est-à-dire \(\ln(ab)=\ln a+\ln b\). \(\blacksquare\)
Démonstration — ln(aⁿ) = n·ln(a)
Pour \(n\geq 1\) : par récurrence. \(n=1\) : trivial. Si vrai au rang \(n\) :
\(\ln(a^{n+1})=\ln(a^n\cdot a)=\ln(a^n)+\ln a=n\ln a+\ln a=(n+1)\ln a\). \(\checkmark\)

Pour \(n=0\) : \(\ln(a^0)=\ln 1=0=0\cdot\ln a\). \(\checkmark\)

Pour \(n<0\) : \(\ln(a^n)=\ln\!\left(\dfrac{1}{a^{-n}}\right)=-\ln(a^{-n})=-(-n)\ln a=n\ln a\). \(\blacksquare\)

Applications — simplifications

• \(\ln 6 = \ln 2 + \ln 3\)

• \(\ln\!\left(\dfrac{1}{2}\right) = -\ln 2\)

• \(\ln 8 = \ln 2^3 = 3\ln 2\)

• \(\ln\!\sqrt{5} = \dfrac{1}{2}\ln 5\)

• \(\ln e^{-3} = -3\)

• \(e^{2\ln 3} = e^{\ln 3^2} = 3^2 = 9\)

Résolution d'équations

\(\ln(f(x)) = \ln(g(x)) \iff f(x) = g(x)\) (sous réserve que \(f(x)>0\) et \(g(x)>0\)).
\(\ln(f(x)) = k \iff f(x) = e^k\) (sous réserve \(f(x)>0\)).
\(\ln(f(x)) > \ln(g(x)) \iff f(x) > g(x) > 0\) (car \(\ln\) est croissante).

04

Dérivée de ln et de ln(u)

\[(\ln x)' = \frac{1}{x}\quad (x>0)\] \[(\ln u)' = \frac{u'}{u}\quad (u>0)\] \[(\ln |u|)' = \frac{u'}{u}\quad (u\neq 0)\]

Exemples de dérivation

• \(f(x)=\ln(3x+1)\) : \(u=3x+1\), \(u'=3\) → \(f'(x)=\dfrac{3}{3x+1}\)

• \(f(x)=\ln(x^2+1)\) : \(u=x^2+1\), \(u'=2x\) → \(f'(x)=\dfrac{2x}{x^2+1}\)

• \(f(x)=x\ln x\) : règle du produit → \(f'(x)=\ln x + x\cdot\dfrac{1}{x} = \ln x + 1\)

• \(f(x)=\dfrac{\ln x}{x}\) : règle du quotient → \(f'(x)=\dfrac{\frac{1}{x}\cdot x - \ln x}{x^2}=\dfrac{1-\ln x}{x^2}\)

• \(f(x)=\ln(\ln x)\) : composée → \(f'(x)=\dfrac{1}{x\ln x}\) (\(x>1\))

Primitive de 1/x et de u'/u

\[\int\frac{1}{x}\,\mathrm{d}x = \ln|x| + C\] \[\int\frac{u'}{u}\,\mathrm{d}x = \ln|u| + C\] Ces formules sont fondamentales pour le calcul intégral (programme de Terminale).

Croissance comparée

Pour tout \(n\geq 1\) :

\[\lim_{x\to+\infty}\frac{\ln x}{x^n}=0\qquad\text{(les puissances l'emportent sur ln en }+\infty\text{)}\] \[\lim_{x\to 0^+}x^n\ln x=0\qquad\text{(les puissances l'emportent sur }\ln\text{ en }0^+\text{)}\] Hiérarchie en \(+\infty\) : \(e^x \gg x^n \gg \ln x\)
Démonstration — \(\lim_{x\to+\infty}\dfrac{\ln x}{x}=0\)
On utilise l'encadrement : pour \(x>0\), \(\ln x \leq x-1 < x\).
Donc \(\ln\!\sqrt{x}\leq\sqrt{x}\), soit \(\dfrac{1}{2}\ln x\leq\sqrt{x}\), soit \(\dfrac{\ln x}{x}\leq\dfrac{2}{\sqrt{x}}\).

De plus \(\dfrac{\ln x}{x}\geq 0\) pour \(x\geq 1\). Par le théorème des gendarmes :
\(0\leq\dfrac{\ln x}{x}\leq\dfrac{2}{\sqrt{x}}\to 0\), donc \(\dfrac{\ln x}{x}\to 0\). \(\blacksquare\)
Démonstration — \(\lim_{x\to 0^+}x\ln x=0\)
Substitution \(t=-\ln x>0\) (valide pour \(x\in(0,1)\), \(x\to 0^+\Leftrightarrow t\to+\infty\)) :

\(x\ln x=e^{-t}\cdot(-t)=-te^{-t}\to 0\) car \(\dfrac{t}{e^t}\to 0\) (croissance comparée exp/polynôme). \(\blacksquare\)

Étude complète — exemples types

Fonction \(f(x) = x\ln x\)

Domaine : \(]0;+\infty[\). Prolongement en 0 : \(\tilde f(0)=0\).

Limites : \(\lim_{x\to 0^+}x\ln x=0\) ; \(\lim_{x\to+\infty}x\ln x=+\infty\).

Dérivée : \(f'(x)=\ln x+1\). Signe : \(f'(x)=0\Leftrightarrow\ln x=-1\Leftrightarrow x=e^{-1}=\dfrac{1}{e}\).

\(f'<0\) sur \(]0;1/e[\) (décroissante), \(f'(1/e)=0\) (minimum), \(f'>0\) sur \(]1/e;+\infty[\) (croissante).

Minimum : \(f(1/e)=\dfrac{1}{e}\cdot(-1)=-\dfrac{1}{e}\approx -0{,}368\).

Dérivée seconde : \(f''(x)=1/x>0\) → convexe sur \(]0;+\infty[\).

Point remarquable : \(f(1)=0\), passe par \((1;0)\).

Fonction \(g(x) = \dfrac{\ln x}{x}\)

Domaine : \(]0;+\infty[\).

Limites : \(\lim_{x\to 0^+}\frac{\ln x}{x}=-\infty\) (AV \(x=0\)) ; \(\lim_{x\to+\infty}\frac{\ln x}{x}=0\) (AH \(y=0\)).

Dérivée : \(g'(x)=\dfrac{1-\ln x}{x^2}\). Signe : \(g'=0\Leftrightarrow\ln x=1\Leftrightarrow x=e\).

\(g'>0\) sur \(]0;e[\) (croissante), \(g'(e)=0\) (maximum), \(g'<0\) sur \(]e;+\infty[\) (décroissante).

Maximum : \(g(e)=\dfrac{1}{e}\approx 0{,}368\). Point de passage : \((1;0)\).

Démonstrations exemplaires du programme

Démo 1 — \(\ln(ab)=\ln a+\ln b\)

Voir section 03 — démonstration par dérivée et unicité de la primitive.

Démo 2 — \(\lim_{x\to+\infty}\dfrac{\ln x}{x}=0\)

Pour \(x>0\) : l'inégalité \(\ln t\leq t-1\) (consequence de la concavité de \(\ln\) — voir démo 4) donne en \(t=\sqrt{x}\) :

\(\ln\sqrt{x}\leq\sqrt{x}-1<\sqrt{x}\), soit \(\frac{\ln x}{2}\leq\sqrt{x}\), soit \(0\leq\frac{\ln x}{x}\leq\frac{2}{\sqrt{x}}\to 0\).

Par le théorème des gendarmes : \(\dfrac{\ln x}{x}\to 0\). \(\blacksquare\)

Démo 3 — \(\lim_{x\to 0^+}x\ln x=0\)

Substitution \(t=-\ln x\) (donc \(x=e^{-t}\), \(x\to 0^+\Leftrightarrow t\to+\infty\)) :

\(x\ln x = e^{-t}\cdot(-t) = -t e^{-t}\)

Or \(\lim_{t\to+\infty}te^{-t}=\lim_{t\to+\infty}\frac{t}{e^t}=0\) par croissance comparée exp/polynôme.

Donc \(x\ln x = -te^{-t}\to 0\). \(\blacksquare\)

Démo 4 — Inégalité fondamentale : \(\ln x\leq x-1\)

Posons \(h(x)=x-1-\ln x\) pour \(x>0\).
\(h'(x)=1-\dfrac{1}{x}=\dfrac{x-1}{x}\).
\(h'<0\) sur \(]0;1[\) (décroissante), \(h'(1)=0\), \(h'>0\) sur \(]1;+\infty[\) (croissante).
Donc \(h\) a un minimum en \(x=1\) : \(h(1)=0\).
Ainsi \(h(x)\geq 0\) pour tout \(x>0\), soit \(\ln x\leq x-1\). \(\blacksquare\)

Conséquences : En remplaçant \(x\) par \(1/x\) : \(-\ln x\leq\frac{1}{x}-1\), soit \(\ln x\geq 1-\frac{1}{x}\).
Donc : \(1-\dfrac{1}{x}\leq\ln x\leq x-1\) (encadrement de \(\ln x\)).

Démo 5 — Dérivée de \(\ln u\) : \((\ln u)'=u'/u\)

Application directe de la règle de la chaîne : \(\ln u = \ln\circ\, u\).
\((\ln x)' = 1/x\), donc \((\ln u)' = u'\cdot\dfrac{1}{u} = \dfrac{u'}{u}\). \(\blacksquare\)
